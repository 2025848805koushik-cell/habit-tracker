# Ultimate Habit Tracker

A clean, interactive habit tracking web app built with HTML, CSS, and JavaScript.

## Features
- Add and complete daily habits
- Monthly calendar with color-coded days
- Progress rings (Daily / Weekly / Monthly)
- Analytics charts
- Weekly summary
- Dark / Light theme
- Export data to CSV
- Fully responsive (mobile friendly)

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
- GitHub Pages

## Live Demo
https://2025848805koushik-cell.github.io/habit-tracker/

## How to Use
1. Add a habit
2. Click to mark done
3. Track progress via calendar & charts
4. Export data anytime
